What is this?

This program is a calculator I made within python that has fully functional GUI interface as well as being integrated with the keyboard for ease of access.

How do I run it?

Here's an option!

1. Download VSCode and get the extension "Python"
2. Open "CalcMain" in VSCode 
3. Open the folder "CalcMain" is stored in into the VSCode file explorer
4. Install PySimpleGui through pip
5. Run the program using the play button in the top right
